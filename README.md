# 🏆 Minions MC PVP Leaderboards

Ένα σύγχρονο και δυναμικό **PvP Leaderboard** για Minecraft servers. Το project αυτό παρακολουθεί τις επιδόσεις των παικτών σε διάφορα kits και υπολογίζει αυτόματα ένα συνολικό σκορ (Overall Skills).

## 🚀 Λειτουργίες

* **Dynamic Leaderboard:** Αυτόματος υπολογισμός πόντων βάσει κατάταξης (5 για 1η θέση, 3 για 2η, 2 για 3η και 1 για τις υπόλοιπες).
* **MCTiers API Integration:** Live εμφάνιση των ranks (HT/LT) απευθείας από το επίσημο MCTiers API.
* **NameMC Profile Links:** Άμεση μετάβαση στο προφίλ του παίκτη στο NameMC με ένα κλικ.
* **Interactive Tooltips:** Αναδυόμενα παράθυρα με UUID, συνολικούς πόντους και rank κατά το hover.
* **Premium Design:** UI με neon glows, custom χρώματα ανά kit και ειδικό badge για τον **PVP MASTER**.

## 🛠️ Τεχνολογίες

Το project είναι εξαιρετικά ελαφρύ και **δεν απαιτεί καμία εγκατάσταση**:
* **HTML5 / CSS3:** Για τον σχεδιασμό και τα εφέ.
* **Vanilla JavaScript:** Για όλη τη λογική και τα API calls.
* **JSON:** Για την αποθήκευση των δεδομένων των παικτών.
